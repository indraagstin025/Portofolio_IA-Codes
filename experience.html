Mendapatkan pengalaman di **Telkom Corporate University** dengan tugas utama memelihara server, memasang Access Point, dan mengelola jaringan Mikrotik, termasuk konfigurasi DHCP server dan switch.

Project: Online Library (2023)

Membangun website **Online Library**, sebuah sistem peminjaman buku menggunakan **Laravel 10**.

Project: Digital Signature (2024)

Mengembangkan proyek **Digital Signature** menggunakan framework **Python Flask**, yang berfungsi untuk membuat tanda tangan digital.

Project: HR System (2025)

Membangun **HR System** dengan backend menggunakan **Fiber GO** (REST API) dan frontend **HTML, CSS, JavaScript**. Fitur utama mencakup absensi QR Code, manajemen jadwal, pengajuan izin, dan slip gaji karyawan.

Project: Moodvis

Mengembangkan **Moodvis**, sebuah website deteksi emosi dengan **Lumen PHP** dan **Face API.js**. Aplikasi ini memiliki fitur deteksi emosi dari webcam serta laporan emosi mingguan/bulanan yang dapat diekspor ke PDF.
